# Artifact Provenance

`.shortcut` files are release artifacts, not source. Releases should include metadata with:

- Manifest snapshot.
- Security audit output.
- Source commit.
- Adapter build metadata.
- SHA-256 checksums for every artifact.

Strict audit reports missing checksum sidecars for packaged artifacts.
