# shortcuts

`shortcuts` makes Apple Shortcuts and App Intents maintainable: typed manifests, validation, security review, generated catalog pages, tests, and release provenance.

## Start here

- Author source-first shortcut packages.
- Validate `shortcut.yml` with `shortcutkit`.
- Review declared permissions and detected capabilities.
- Generate catalog and docs references.
