# Shortcut Contributions

Before opening a pull request:

- Add `shortcut.yml` with declared permissions.
- Include source under `src/` and a package `README.md`.
- Do not commit `.shortcut` artifacts outside `dist/`.
- Run validate, lint, catalog check, and strict audit.
- Explain any network, AI/model, shell, or URL scheme behavior.
