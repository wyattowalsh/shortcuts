# Known Limitations

- `.shortcut` decompilation is not implemented.
- Cherri and Jelly adapters require external binaries and are not run by default.
- Runtime tests require macOS and Apple's `shortcuts` CLI.
- Security audit is source-signal based and does not claim absolute safety.
- App Intents examples are snippets, not a signed demo app.
