# App Intents Lab

The `app-intents/` package contains Swift examples for:

- Minimal parameterized `AppIntent` implementations.
- `AppShortcutsProvider` phrase design.
- `AppEnum` modeling for constrained parameters.
- Availability notes for iOS 18 and macOS 15.

Treat these examples as copyable snippets for app projects and validate with Xcode on macOS.
