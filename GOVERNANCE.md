# Governance

`shortcuts` is governed by maintainers who prioritize source-first shortcut development, honest security review, reproducible release metadata, and practical App Intents education.

## Decision Principles

- The manifest is the contract.
- Source is canonical; artifacts are generated or fixtures.
- Unknown risk is explicit risk.
- Optional adapters stay external-process based and license-aware.
- macOS runtime behavior is gated and never claimed unless actually run.

Major behavior changes should be represented in OpenSpec before implementation.
