# Roadmap

## Beta

- Source-first shortcut packages with typed manifests.
- `shortcutkit` validation, linting, catalog generation, and strict security audit.
- Fumadocs documentation with generated catalog and LLM text routes.
- Adapter capability matrix with manual, artifact, Cherri, and Jelly modes.
- Optional macOS runtime testing behind explicit flags.

## Next

- Broader ActionDB coverage from real shortcut corpora.
- Signed release artifacts and SBOM/provenance attestations.
- More App Intents examples and Xcode validation workflows.
- Contributor-submitted shortcut catalog entries after maintainer review.
